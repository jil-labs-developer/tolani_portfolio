# tolaniadekoya.com

React 18 + TypeScript + Vite, using React Router v6, TanStack Query, Framer
Motion, and Tailwind CSS v4 (`@tailwindcss/vite`). 

## Commands

```bash
npm install
npm run dev        # localhost:8935
npm run build      # writes dist/ (Vercel's default Output Directory)
npm run preview    # serve the build on localhost:8935
npm run typecheck  # tsc --noEmit
```
